# sales-e-commerce
Interactive Tableau dashboard analysing e-commerce sales, profit, quantity, weekly trends, and product category performance.
E-Commerce Sales & Profit Dashboard | Tableau
Project Overview

This project features an interactive Tableau dashboard built to analyse e-commerce sales performance. The dashboard focuses on key business KPIs, weekly trends, and product category insights to support data-driven decision making.

Key KPIs

Total Sales

Total Profit

Total Quantity Sold

These KPIs provide a high-level overview of overall business performance.

Trend Analysis

Weekly sales trends

Weekly profit trends

Weekly aggregation helps identify short-term changes and performance patterns over time.

Product Category Analysis

The dashboard analyses performance across the following categories:

Chairs

Phones

Tables

Binders

Storage

Bookcases

Accessories

Furnishings

This enables comparison of sales, profit, and quantity across product groups.

Tools Used

Tableau: data visualisation and interactive dashboard development

Dashboard Purpose

The dashboard is designed to help stakeholders:

Monitor overall sales and profitability

Track weekly performance trends

Identify high and low performing product categories

Support strategic and operational decisions

What This Project Demonstrates

KPI development for business reporting

Time-based trend analysis in Tableau

Clear dashboard design and storytelling

Translating raw data into actionable insights
